# Atom

My Atom.io Config

You can download Atom Editor from http://atom.io

You need to install these packages:

+ atom-monokai
+ ex-mode
+ file-icons
+ jump-to-line
+ jumpy
+ language-latex
+ latexer
+ markdown-writer
+ minimap
+ monokai
+ tidy-markdown
+ vim-mode
+ vim-mode-visual-block
+ vim-surround

And config keymap.cson

Last you need to need to set
UI Theme: One Dark
Syntax Theme: Monokai
